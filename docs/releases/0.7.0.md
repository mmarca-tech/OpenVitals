# OpenVitals 0.7.0

## English

OpenVitals 0.7.0 adds route-based activity logging and the beta-first release flow for public testing.

- Add Activity entry support for Health Connect exercise sessions, routes, distance, elevation gain, active calories, and total calories.
- Import GPX, KML, and KMZ routes, preview them, infer activity details, retime untimestamped tracks, and save route-backed workouts to Health Connect.
- Record GPS activities from OpenVitals with pause, resume, discard, route preview, distance, elevation gain, moving time, and review before saving.
- Show a persistent recording notification while GPS recording is active so the user can see that an activity is being recorded.
- Estimate active and total calories for imported routes and recorded activities while leaving fully manual activity entries blank.
- Publish release tags to Codeberg as beta prereleases and to Google Play open testing first, then promote the approved build to stable Codeberg and Play production.

## Espanol

OpenVitals 0.7.0 anade registro de actividades con ruta y un flujo de lanzamiento beta primero para pruebas publicas.

- Anade entradas de Actividad para sesiones de ejercicio de Health Connect, rutas, distancia, elevacion, calorias activas y calorias totales.
- Importa rutas GPX, KML y KMZ, las previsualiza, infiere detalles de actividad, ajusta rutas sin marcas de tiempo y guarda entrenamientos con ruta en Health Connect.
- Graba actividades GPS desde OpenVitals con pausa, reanudacion, descarte, previsualizacion de ruta, distancia, elevacion, tiempo en movimiento y revision antes de guardar.
- Muestra una notificacion persistente durante la grabacion GPS para que el usuario vea que se esta registrando una actividad.
- Estima calorias activas y totales en rutas importadas y actividades grabadas, dejando vacias las actividades completamente manuales.
- Publica las etiquetas de version en Codeberg como beta y en pruebas abiertas de Google Play primero; despues promociona la compilacion aprobada a estable en Codeberg y produccion en Play.
