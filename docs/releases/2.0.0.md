# OpenVitals 2.0.0

Released 2026-07-11.

OpenVitals is now built on Flutter. Same app, same features, same Health Connect data - and the foundation for an iOS version later.

- **App settings reset with this update.** Goals, custom drinks, the caffeine profile, paired sensors, reminders, and offline map packs need to be set up again. Your Health Connect data is stored outside the app and is NOT affected.
- Apple Health imports are much faster on large exports: records in categories you did not select are skipped before they are ever read, and progress now advances while the export is being scanned instead of sitting at 0%.
- Fixed missing heart rate at the start of a recorded activity.
- Sanitized diagnostics logs can be shared directly from Settings.

The app keeps the same package name and signing certificate, so this installs as a normal update.
