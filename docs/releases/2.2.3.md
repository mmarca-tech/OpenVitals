# OpenVitals 2.2.3

Released 2026-07-13.

The last of the Google out of the build.

### Changed

- **Google's dependency blob is no longer stamped into the APK.** Android's build tools quietly attach a block to every APK they sign: a list of every library the app was built from, encrypted with a Google public key so that only the Play Store can read it. Nobody else — not you, not F-Droid, not us — can see what it says. It survived 2.2.1, which removed Google Play Services, because it is not a dependency at all: it is added at the moment of signing, after everything else is done. It is gone from the APK now. The app bundle we upload to Play keeps it, because that is the one place it is actually read.

Nothing else changed. There is no behavioural difference between 2.2.2 and 2.2.3; the release exists because F-Droid will not distribute an APK carrying an opaque Google block, and it was the last thing standing between the two.
