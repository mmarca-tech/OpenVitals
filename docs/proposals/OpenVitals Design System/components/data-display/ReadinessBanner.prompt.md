**ReadinessBanner** — the hero card on the Daily Readiness screen: accent-tinted, with a leading chip, title + confidence line, a large score, an accent headline, and a body explanation. Pass sub-tiles as children.

```jsx
<ReadinessBanner score="65/100" confidence="Medium confidence · sleep data missing"
  headline="Train, but keep it controlled"
  body="Do moderate training today, but avoid maximal effort.">
  {/* Body Energy / Training Readiness tiles */}
</ReadinessBanner>
```
