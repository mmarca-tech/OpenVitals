# How-to Guides

Step-by-step usage for each feature now lives in a **"How to use it"** section at the top of that feature's page under [Features](../features/README.md) — for example [Body Energy](../features/body-energy.md#how-to-use-it), [Activity recording](../features/activity-recording.md#how-to-use-it), or [Beverage logging and caffeine](../features/beverage-logging-and-caffeine.md#how-to-use-it). The shared metric-screen controls are documented once in [Statistics](../features/statistics.md#how-to-use-it).

This section holds the longer-form workflows that need more room than a feature page — usually because they involve steps outside the app (preparing files, using a computer, or troubleshooting).

- [Add offline maps](offline-maps.md): download PMTiles or Mapsforge map packs, import them into OpenVitals, choose the active render format, and troubleshoot common issues.
